Detailed Description

Controls a simulation run initiated with the command line interface (cli).

CliController functions include:

  • accepts the commandline arguments
  • checks the OpenMP environment
  • checks the file system environment
  • reads the config file specified on the cli
  • effects cli overrides of config parameters
  • patches the config file for any remaining defaults
  • interprets and executes the output prefix
  • makes a stride logger The CliController execution
  • creates a simulation runner (
    See Also
    SimRunner)
  • registers the appropriate viewers
  • runs the simulation
